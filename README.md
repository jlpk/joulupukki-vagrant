===================
Joulupukki Vagrant
===================

Deploy Joulupukki with Vagrant


Prerequisite
============

* Vagrant 1.7.2
* Virtualbox

Installation
============

::

  git clone https://github.com/jplk/joulupukki-vagrant
  cd joulupukki-vagrant
  vagrant up

===================
Joulupukki Vagrant
===================

Deploy Joulupukki with Vagrant


Prerequisite
============

* Vagrant 1.7.2
* Virtualbox

Installation
============

```bash
    git clone https://github.com/jplk/joulupukki-vagrant
    cd joulupukki-vagrant
    vagrant up 
    vagrant ssh 
    cd joulupukki-web && npm start
```

Use 
===

    * using box: ubuntu/trusty64
    * grunt
    * virtualenv
    * pip
    * npm
    * git 
    * ruby
    * ruby-sass
